# artifacts

Documentação orientada a **humanos**: guias, decisões de design, diagramas e notas.

> O `CLAUDE.md` na raiz é o contrato voltado ao agente de código; esta pasta é para as pessoas.

## Convenções

- **Relatórios de desenvolvimento** (`development-reports/`): um arquivo por entrega, nomeado
  `YYYY-MM-DD-HHmm-<slug>.md` (data **e hora**, 24h), para que ordenem cronologicamente — ex.:
  `2026-06-15-2124-crm-lead-qualification.md`.
